# 云函数目录

本目录由 `manifest.json` 的 `cloudfunctionRoot` 指向。

当前日程数据使用**客户端直连云数据库**（`schedules` 集合）。

## 云数据库 `schedules`

- 集合名称：`schedules`
- 环境 ID：见 `src/config/weixin-cloud.ts`

### 权限（重要）

若在**云开发控制台**里手动添加/导入数据，小程序里却拉不到，多半是权限为 **「仅创建者可读写」**：

- 控制台录入的记录**没有当前用户的 `_openid`**
- 小程序端 `collection.get()` **只能看到自己用 App 创建的数据**

**个人使用（仅自己一个账号）** 建议改为：

- **所有用户可读，仅创建者可写**

这样控制台批量导入的数据，小程序也能读到。

若必须「仅创建者可读写」，请只在**小程序里新增/编辑**，不要用控制台导数据。

### 字段要求

每条记录需包含（与 `ScheduleItem` 一致）：

| 字段 | 必填 | 说明 |
|------|------|------|
| `id` | 建议 | 业务主键，如 `"1"`；无则用云 `_id` |
| `date` | 是 | `YYYY-MM-DD` |
| `time` | 否 | 默认 `09:00` |
| `title` | 是 | 标题 |
| `category` | 否 | 默认 `其他` |
| `categoryIcon` | 否 | 默认 `📌` |

### 拉取条数

客户端已做**分页拉取**（突破默认每次 20 条限制）。下拉首页可强制同步云端。
